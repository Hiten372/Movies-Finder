# 🎬 Movie Finder

Movie Finder is a modern and responsive web application that allows users to search for movies using the OMDb API. Users can search for any movie and instantly view movie posters, titles, release years, and movie types in a clean and interactive interface.

This project is built using HTML, CSS, and JavaScript and demonstrates real-world frontend concepts such as API integration, async/await, DOM manipulation, event handling, dynamic rendering, and error handling.

---

# 🚀 Features

- Search movies by name
- Fetch real-time movie data from OMDb API
- Display movie posters and details
- Responsive movie card layout
- Loading state while fetching data
- Error handling for invalid searches
- Fallback image for unavailable posters
- Clean and modern UI design

---

# 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- OMDb API
